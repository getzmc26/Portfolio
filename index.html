Aerial Attack

This HTML canvas side-scrolling game was written using vanilla JavaScript, AJAX, XML, PHP, and SQL. The user plays as a jet that avoids colliding with other jets. The game implements a full collision system, health system, projectile system, and uses multiple threads to keep all of the action moving. If the user makes it within the current top ten high scores, then the database is updated accordingly with the score. I even created the art assets for the game, made the sound effects, and wrote the music!
